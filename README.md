# QR Tool No Ads

A simple QR code generator built for speed, convenience, and a clean experience — with no ads.

## Website

[https://lahieuphong.github.io/qr-tool-no-ads/](https://lahieuphong.github.io/qr-tool-no-ads/)

## Features

- Generate QR codes from URLs
- Automatically add `https://` when needed
- Download as PNG or SVG
- Print QR codes
- Customize QR color, background, size, and margin
- Choose error correction level
- Dark mode / light mode
- Vietnamese / English interface
- Save recent links in local storage
- No ads

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- qrcode

## Run Locally

```bash
yarn install
yarn dev
````

## Build

```bash
yarn build
```

## Notes

* Built with static export for GitHub Pages
* Main config files: `next.config.ts`, `src/lib/app-config.ts`
* User settings and recent links are stored in browser local storage
